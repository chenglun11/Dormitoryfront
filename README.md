### Requirements
本项目基于Layui+ JWT + Axios
请注意区分Layui版本为2.7x

本项目前后端分离,且区分repo.
请务必通过 [后端仓库地址](https://github.com/chenglun11/DormitoryCMS) 打开


### Getting Started
1. Clone本项目并从WebStorm打开
2. 使用IDEA自带的Maven管理器安装依赖
3. 将SQL导入数据库(其中测试数据自行处理)
4. 运行✅

### Version Control
